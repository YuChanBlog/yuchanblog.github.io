搭建HEXO

配置GitHub

ssh-keygen -t rsa -C "你的注册github的电子邮箱地址"

一直回车，然后在本地C盘-用户目录下面有一个.ssh文件夹，进去打开id_rsa.pub。复制内容放进GitHub上的settings里的ssh keys中。然后在git bash再输入：

ssh -T git@github.com

如果如果是第一次的会提示是否continue，输入yes就会看到：You’ve successfully authenticated, but GitHub does not provide shell access 。这就表示已成功链接。

链接GitHub与hexo：

打开_config.yml文件在行末添加内容：

deploy:  
    type: git  
    repository: git@github.com:xpf199741/xpf199741.github.io.git   
    branch: master

最后git bash执行：

hexo g  
hexo d

问题

如果出现 ERROR Deployer not found: git时，git bash输入

npm install hexo-deployer-git --save
